# WP SMTP Automation Tool
**Author [Aldo Borghes](https://www.linkedin.com/in/aldo-borghes/) 

Este repositorio contiene las herramientas necesarias para automatizar la configuración de SMTP en instancias Bitnami WordPress utilizando Amazon SES.

## Características
- Implementación vía MU-Plugins para evitar desactivaciones accidentales.
- Gestión segura de credenciales mediante archivos `.env` fuera del DocumentRoot.
- Scripts de automatización para despliegues rápidos.
